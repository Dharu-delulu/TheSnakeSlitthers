# TheSnakeSlitthers
Building Snake on an Arduino Nano was "destiny." A piezo buzzer flyback fried my USB port, but I didn't stop. I bypassed the dead port using an UNO via UART—turning a fail into a masterclass in serial comms.  Optimized with: int8_t snake arrays (RAM saver) 300-700 joystick deadzones  SH1106 OLED (U8g2)
